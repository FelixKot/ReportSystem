---Установка:
= Вставить в конец мода перехватчики из Перехватчики.txt.
= Отключить стандартный репорт (если требуется) с помощью /* */
= Скомпилировать мод.
= Скопировать папки filterscripts и scriptfiles на сервер.
= Настроить report_settings.ini и report.phrases_*.txt (если требуется).
= Скопировать файлы из plugins на сервер (если требуется или если отсутствуют).
= В server.cfg прописать filterscripts reportsystem_* и плагины (если добавляли) и перезагрузить сервер.

[!] Внимание! Если вы хотите получать обновления скрипта: обязательно измените значение root в report_settings.ini


---Доп.функции скрипта:
= Если требуется вызвать диалог репорта через мод:
- В начало мода добавить: #define RSDT(%0) CallRemoteFunction("RSDT", "i", %0)
- Далее в нужном месте для вызова используйте RSDT(playerid);
---
= PVar для проверки кол-ва ответов админа за сессию:
- GetPVarInt(playerid, "ReportAnsed")
- Пример использования:
	if(сюда проверку на админку && GetPVarInt(playerid, "ReportAnsed") > 0)
 	{
		тут запись данных из GetPVarInt(playerid, "ReportAnsed")
 	}
---
= Функция очистки репортов игрока. Если требуется удалить репорты, например, при выдаче игроку мута. (Важно! При OnPlayerDisconnect используется автоматически!)
- В начало мода добавить: #define RSCPR(%0) CallRemoteFunction("RSCPR", "i", %0)
- Для вызова используйте RSCPR(playerid); //playerid - идентификатор игрока, чьи репорты необходимо очистить.
